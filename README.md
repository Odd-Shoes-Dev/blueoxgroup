# Blue Ox Group
 
Website for Blue Ox Group: showcases the group and its member software
companies, and hosts a directory of sales/marketing representatives (profile,
contact details, active/inactive availability).

## Structure

- `frontend/` — Next.js app (App Router, TypeScript, Tailwind, shadcn/ui). The
  only running application right now.
- `backend/` — reserved placeholder for a future Express service. Not yet implemented.
- `database/migrations/` — SQL migrations, applied manually to Neon.
- `docs/` — architecture, schema, provider abstraction layer, setup instructions,
  and notes on deferred future scope.

See [`docs/setup.md`](docs/setup.md) to run locally, and
[`docs/architecture.md`](docs/architecture.md) for how the codebase is organized.
